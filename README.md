# Maintenance AI Planner - Vercel Ready

هذه نسخة جاهزة للنشر على Vercel كمشروع Next.js.

## طريقة الاستخدام
1. فك الضغط عن الملف.
2. ارفع الملفات كلها إلى GitHub داخل المستودع.
3. في Vercel اختر Import Project.
4. اختر هذا المستودع.
5. Application Preset: Next.js.
6. اضغط Deploy.

## ملاحظات مهمة
- هذه النسخة لا تربط الإيميل حتى الآن.
- البيانات تحفظ في متصفحك عبر localStorage.
- المرحلة التالية: ربط Supabase Free لجعل البيانات تظهر من أي جهاز.
